# Dai Thanh Furniture — Website mới

Website Next.js cho **DAI THANH FURNITURE JSC**, lấy nội dung từ [daithanhfurniture.com](https://daithanhfurniture.com/) và định hướng thẩm mỹ theo phong cách premium của [atc-craft.com](https://atc-craft.com/).

## Công nghệ

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- Framer Motion
- SEO: Metadata API, sitemap, robots, JSON-LD

## Chạy dự án

```bash
npm install
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000).

## Trang

| Route | Nội dung |
|-------|----------|
| `/` | Hero, story, collections, products, values, catalogue CTA, news, contact |
| `/about` | Giới thiệu công ty |
| `/products` | Danh mục sản phẩm + filter |
| `/products/[slug]` | Chi tiết sản phẩm |
| `/factory` | Dây chuyền nhà máy |
| `/contact` | Form liên hệ + thông tin |

## Brand

Logo nằm trong `public/logo-mark.png` và `public/logo-full.png`. Palette vàng–đồng trên nền tối theo identity Dai Thanh.
